## 🎯 Mục tiêu

- Giúp người Việt tiếp cận AI dễ hiểu, dễ dùng, dễ lan tỏa – đặc biệt là học sinh, người lao động, người không rành công nghệ.
- Tạo công cụ khơi gợi sự tò mò & cảm hứng khám phá AI lần đầu.
- Mint NFT “Tôi đã dùng AI lần đầu” để ghi dấu hành trình trải nghiệm của từng người.

## ✨ Tính năng MVP hiện tại

- 🌟 Trang chủ giới thiệu chiến dịch AI ƠI!
- 💡 `PromptGroup.tsx`: Gợi ý prompt theo nhóm người dùng (học sinh, mẹ bán hàng, chú xe ôm)
- 📝 `TryAIForm.tsx`: Ghi nhận cảm xúc “Tôi vừa dùng AI lần đầu”
- 🎁 NFT: Tùy chọn mint chứng nhận AI lần đầu từ nền tảng NFTơi!

## ▶️ Cách chạy dự án local

```bash
git clone https://github.com/quang06091988/ai-oi-app.git
cd ai-oi-app
npm install
npm run dev
```

Truy cập: [http://localhost:5173](http://localhost:5173)
